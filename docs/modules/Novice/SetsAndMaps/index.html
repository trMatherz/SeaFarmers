Maps

Maps are a lot like arrays, except that their indices are not 0−n0-n0−n, then can be anything you want them to be!

A map takes a data type, like int or string for BOTH its keys and values. These can be different. This means that you can get a value of a int from a string or a boolean from an int.

map<int, int> a; 
if(a.count(3)) cout << a[3] << endl;
else cout << "NONE" << endl;
a[3] = 90990; 
if(a.count(3)) cout << a[3] << endl;
else cout << "NONE" << endl;

Note: By nature of the maps, accessing elements takes O(log⁡2(n))\mathcal{O}(\log_2(n))O(log2​(n))

Sets

Sets don't have indices but they contain all elements in a sorted order.

You can insert and erase values in O(log⁡2(n))\mathcal{O}(\log_2(n))O(log2​(n))

set<int> a; 
a.insert(2); 
a.insert(3); 
cout << a.size() << endl; // 2
a.insert(2); 
cout << a.size() << endl; // 2 since this does nothgin
a.erase(2); 
cout << a.size() << endl; // 1
