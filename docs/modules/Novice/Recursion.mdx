Now, we can take these functions, which may seem elementary right now, and spice them up by having functions call 
themselves. Look at the code below: 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`int count = 0; 
void increaseCount(int x) {
    if(x <= 0) return; 
    count++; 
    increaseCount(x - 1); 
}

int main() {
    cout << count << endl;
    increaseCount(1); 
    cout << count << endl;
    increaseCount(3); 
    cout << count << endl;
}
`}
  </SyntaxHighlighter>
</div>

This code and increases the count variable by the initial $x$ value. Though this code may seem extreme to solve such a 
simple problem, think about how the code runs.

Inside the function, there is a base case: This is when $x \leq 0$. Since count doesn't need to be increased anymore, 
the funciton returns and does nothing. 

Else: The function increases count by $1$ and then calls increaseCount($x - 1$). 

To really break it down, increaseCount($x$) has the same affect as increaseCount($1$), increaseCount($x - 1$). With 
increaseCount($1$) only increasing count once.
