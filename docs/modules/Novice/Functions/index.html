An Introduction to Functions

Now that you have some experience solving basic problems, you can learn how to implement more complex ideas. To do this, we can take the tools you have used to solve "mini" problems in a problem. We can solve these, without copy and pasting code by using functions. Here is an example of a program using a basic function:

int count = 0; 
void increaseCount() {
  count++; 
}

int main() {
  cout << count << endl;
  increaseCount(); 
  cout << count << endl;
  increaseCount(); 
  increaseCount(); 
  cout << count << endl;
}

Observe how you can "call" a function - this makes the program stop where it is and run all of the code in the function.

Try to see how xxx can be used at A, B, and C; yyy can be used at B and C; and zzz can only be used at C.

More Complex Functions

But that's not all: Functions can both RECEIVE and RETURN input!

#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
#include <fstream>
#include <iostream> 
#include <algorithm>
using namespace std;


bool checkX(int x) {
  if(x > 5) return true; 
  return false; 
}

int main() {
  int x = 5; 
  int y = 6; 
  if(checkX(x)) cout << "X is Good!" << endl;
  else cout << "X is Bad!" << endl;
  if(checkX(y)) cout << "Y is Good!" << endl;
  else cout << "Y is Bad!" << endl;
}

For example, if you want a function to determine if a number if greater than 555, you can pass xxx as a variable into the function and receive a return value from the function.

The return value can be a data type: "int", "long long", "string", "bool" and goes BEFORE the function name.

Note: Use "void" if the funciton doesn't return anything.

The parameters of a function (the variables that you "pass to it"), are named IN ORDER inside the () after the function name. You must give a function all of its parameters in order for it to work.

Last example:

bool checkX(int x, string z) {
  if(z == "CAT") return true; 
  if(x > 5) return true; 
  return false; 
}

int main() {
  int x = 5; 
  if(checkX(x, "CAT")) cout << "X is Good!" << endl;
  else cout << "X is Bad!" << endl;
}

Observe that the variables are passed into the function in the specified order.
