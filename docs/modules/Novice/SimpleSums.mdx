# An Introduction to Prefix Sums

If you want to sum the elements of an array betwen $l$ and $r$, your first instinct is to do this: 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`int sum = 0; 
for(int i = l; i <= r; i++) sum += a[i]; 
cout << sum; 
`}
  </SyntaxHighlighter>
</div>

This runs in$\mathcal{O}(n)$

Instead, we can do pre-calculations for the sum of the first $i$ elemenets for each $i$. 

Then, the sum is equal to $pre[r + 1]$ - $pre[l]$ (one-indexed). 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`
vector<int> pre(n + 1, 0); 
for(int i = 0; i < n; i++) pre[i + 1] = pre[i] + a[i]; 
cout << pre[r + 1] - pre[l]; 
`}
  </SyntaxHighlighter>
</div>

This runs in $\mathcal{O}(1)$
