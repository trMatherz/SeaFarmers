# An Introduction to Functions

Now that you have some experience solving basic problems, you can learn how to implement more complex ideas. To do this, 
we can take the tools you have used to solve "mini" problems in a problem. We can solve these, without 
copy and pasting code by using of functions. Here is an example of a program using a basic function:

# More Complex Functions

But that's not all: Functions can both RECEIVE and RETURN input!

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
#include <fstream>
#include <iostream> 
#include <algorithm>
using namespace std;


bool checkX(int x) {
  if(x > 5) return true; 
  return false; 
}

int main() {
  int x = 5; 
  int y = 6; 
  if(checkX(x)) cout << "X is Good!" << endl;
  else cout << "X is Bad!" << endl;
  if(checkX(y)) cout << "Y is Good!" << endl;
  else cout << "Y is Bad!" << endl;
}
`}
  </SyntaxHighlighter>
</div>

For example, if you want a function to determine if a number if greater than $5$, you can pass $x$ as a variable 
into the function and receive a return value from the function. 

The return value can be a data type: "int", "long long", "string", "bool" and goes BEFORE the function name. 

Note: Use "void" if the funciton doesn't return anything. 

The parameters of a function (the variables that you "pass to it"), are named IN ORDER inside the () after the 
function name. You must give a function all of its parameters in order for it to work. 

Last example:
