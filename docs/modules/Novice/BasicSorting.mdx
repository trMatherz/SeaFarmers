# In Competetive Programming, you need not know, at least for now, how to actually implement a sorting function. You can 
use built in systems to do this for you. 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`vector<int> a = {3, 2, 1, 4}; 
sort(a.begin(), a.end()); // a = {1, 2, 3, 4}
`}
  </SyntaxHighlighter>
</div>

The code above sorts the entire vector. The default for sorting is in non-decreasing order. This is useful to a lot of
problems, so don't worry about using cutsom comparators. 

Keep in mind, sorting a vector of size $n$ takes $\mathcal{O}(\log_2(n)*n)$
