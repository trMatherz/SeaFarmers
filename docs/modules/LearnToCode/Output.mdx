# Output

Prerequisites: Nothing

At its core, programs are designed to output something, given an input. For right now, let's keep it simple and 
focus only on outputs. 

This is your basic C++ file: 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <iostream>
using namespace std;

int main() {
    cout << 5 << endl; 
    cout << 2 + 3 << endl; 
}
`}
  </SyntaxHighlighter>
</div>

When this code runs, it starts at the top of main() and goes line by line until it reaches the bottom. The "cout" 
means output, the "\<\<" seperarates different outputs, and the "endl" makes the program output on the next line. 

If you ran the code, you would get an output like this: 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`5
2
`}
  </SyntaxHighlighter>
</div>
