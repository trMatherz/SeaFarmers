Conditionals

Prerequisites: Basic Data Types

Focus Problem

Conditional is a fancy way of saying IF this, THEN this.

if(3 > 4) { //if this is true
b = b + 2; //then this happens
} else if(b > 2) { //if this is true AND the previous if is not true
b = b - 2; //then this happens
} else if(b == 2) { //use 2 equals signs to check if it is the same
  b = 1; 
} else { //if nothing above it is true
  cout << b; //then this happens
}

Note: An 'if' statement can be stand alone, have a following 'else', and have any number of 'else if' statements in the middle if the 'if' statements are not connect by an else, then they can BOTH happen

Possible Expressions

These are the different things can you put in an If-Statement

a == b //returns true if a is equal to b
a != b //returns true if a is equal to b
a < b //returns true if a is less than b
a > b //returns true if a is greater than b
a <= b //returns true if a is less or equal to than b
a >= b //returns true if a is greater than or equal to b

Logical Operators

These are a way of combing expressions

!(expression) //returns true if the expression is false and false otherwise
(expression) || (expression) //returns true if EITHER expression is true
(expression) && (expression) //returns true if BOTH expressions are true

Concrete Examples

if(5 > 4 || 3 > 4) //this equals (true || false) which is true
if(5 > 4 && 3 > 4) //this equals (true && false) which is false
if(5 > 4 && !(3 > 4)) //this equals (true && true) which is true
