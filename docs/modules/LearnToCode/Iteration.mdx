# While Loops

Iteration is a fancy way of saying repeated actions. In programming, this mainly involves doing something WHILE 
something is true. The code in the {} runs will the expression is true. 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`while(*expression*) {
    a = a + 2; 
    b = b + 3; 
}
`}
  </SyntaxHighlighter>
</div>

Here's a comparison: 

An If-Statement runs a code block ONCE, IF the expression is true. 

A While Loop runs a code block WHILE the expression is true. 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`if(a < 2) { 
    a = a + 2; 
    b = b + 3; 
}
`}
  </SyntaxHighlighter>
</div>

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`while(a < 2) { 
    a = a + 2; 
    b = b + 3; 
}
`}
  </SyntaxHighlighter>
</div>

# For Loops 

A For Loop is just another kind of loop that does pretty much the same thing except they "streamline" the process. 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`for(*variable* ; *expression* ; *runs between loops*) { 
    a = a + 2; 
    b = b + 3; 
}
`}
  </SyntaxHighlighter>
</div>

The below code does the same thing. Most of the time, For Loops are more efficient, but you can use While Loops if
you want. 

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`int sum = 0, a = 1; 
while(a <= 5) {
	sum += a; 
    a = a + 1; 
}
cout << sum;
`}
  </SyntaxHighlighter>
</div>

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`int sum = 0; 
for(int a = 1; a <= 5; a = a + 1){
	sum += a;
}
cout << sum; 
`}
  </SyntaxHighlighter>
</div>
