To Eat Or Not To Eat Solution

Tags: Greedy, Simulation

Breakdown

We maintain bes[f]bes[f]bes[f] where fff is the set of colors, for each set of colors.

There are 333 things that can happen:

111 We can eat the food

222 We can go to dinner

333 We can spit out any food

This runs in O(n)\mathcal{O}(n)O(n).

Implementation

#include <cmath>
#include <cstdio>
#include <vector>
// #include <iostream> 
#include <fstream>
std::ifstream cin ("case8.txt");
std::ofstream cout ("out8.txt");
#include <algorithm>
#include <map>
using namespace std;

map<string, int> food = {{"Red", 0}, {"Blue", 1}, {"Yellow", 2}};
long long const inf = (long long) 1e18 + 7; 
int main() {
  vector<long long> bes(8, -inf); 
  bes[0] = 0; 
  int n; 
  cin >> n; 
  for(int i = 0; i < n; i++) {
      string f; 
      cin >> f; 
      if(food.count(f)) {
          for(int j = 0; j < 8; j++) 
              bes[j | (1 << food[f])] = max(bes[j | (1 << food[f])], bes[j]);
      }
      long long r, c; 
      cin >> r >> c; 
      int cur = 0; 
      for(int i = 0; i < r; i++) {
          string x; 
          cin >> x; 
          cur |= (1 << food[x]); 
      }
      bes[cur] += c; 
      for(int j = 0; j < 8; j++) {
          bes[j] = max(bes[j], bes[j | cur]); 
      }
  }
  cout << bes[0]; 
}
