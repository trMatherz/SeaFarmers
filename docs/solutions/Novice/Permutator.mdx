# Permutator Solution

Tags: Sorting

# Breakdown

The values for each $i$ are already determined since $a$<sub>$i$</sub> is constant and the number of subarrays $i$ is in
is constant. We calculate each of these determined values, sort them, sort $b$, then match them up in reverse. 

This runs in $\mathcal{O}(\log_2(n)*n)$

# Implementation

{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n; cin >> n; 
    vector<long long> a(n), b(n); 
    for(int i = 0; i < n; i++) cin >> a[i]; 
    for(int i = 0; i < n; i++) cin >> b[i]; 
    for(int i = 0; i < n; i++) a[i] = a[i] * (i + 1) * (n - i); 
    sort(b.begin(), b.end()); 
    sort(a.begin(), a.end()); 
    long long an = 0; 
    for(int i = 0; i < n; i++) an += a[i] * b[n - 1 - i]; 
    cout << an; 
}
`}
