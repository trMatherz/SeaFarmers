# Two Knights Solution

Tags: Functions

# Breakdown

For each $k$, we can use the following derived formula in $\mathcal{O}(1)$: 

First, count the number of ways to place $2$ knights. There are $k^2$ options for the first knight and $k^2-1$ for 
the second. Though, we must divide this number by $2$ since we do not care which knight is which. $(1, 1), (5, 5)$ is
the same as $(5, 5), (1, 1)$.

Second, we must subtract the number of ways to make kngihts attack each other. In a $2x3$ box, there are $2$ ways to 
place kngihts so that they attack each other. There are $(k - 2) * (k - 1)$ to place this box. The case for the $3x2$ 
box is similar. 

This runs in $\mathcal{O}(n)$
