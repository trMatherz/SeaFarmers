# Sleepy Cow Hearding Solution

Tags: 2P

# Breakdown

We use two pointers to implement the following greedy strategy: It is always better to pair the current 
lowest with the current highest. 

This runs in $\mathcal{O}(n)$

# Implementation

{`#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
std::ifstream cin("pairup.in");
std::ofstream cout("pairup.out");
#include <algorithm>
using namespace std;

int main() {
    int n; cin >> n; 
    vector<pair<int, int>> a(n); 
    for(int i = 0; i < n; i++) cin >> a[i].second >> a[i].first; 
    sort(a.begin(), a.end()); 
    int an = 0; 
    int l = 0, r = n - 1; 
    while(l <= r) {
        int use = min(a[l].second, a[r].second); 
        emax(an, a[l].first + a[r].first); 
        a[l].second -= use; 
        if(r > l) a[r].second -= use; 
        if(a[l].second == 0) l++; 
        if(a[r].second == 0) r--; 
    }
    cout << an; 
}
