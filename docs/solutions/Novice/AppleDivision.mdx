# Apple Division Solution

Tags: Recursion

# Breakdown

We can use recursion to completely search all possible partitions of splitting the apples into two groups.
Our base case
is when we have no more apples. Other than that, we maintain the current sum of the two piles. 

This runs in $\mathcal{O}(2^n)$

# Implementation

{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;

int n;
vector<long long> a; 
long long go(int x, long long sum1, long long sum2) {
    if(x == n) return abs(sum1 - sum2); 
    return min(go(x + 1, sum1 + a[x], sum2), go(x + 1, sum1, sum2 + a[x])); 
}
int main() {      
    cin >> n; 
    a = vector<long long>(n); 
    for(int i = 0; i < n; i++) cin >> a[i]; 
    cout << go(0, 0, 0); 
}
