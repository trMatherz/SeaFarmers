# Distinct Numbers Solution

Tags: Sorting

# Breakdown

If we sort the list, then the answer is equal to the number of elements minus the number of elements where 
$a$<sub>$i$</sub> $=$ $a$<sub>$i+1$</sub>

This runs in $\mathcal{O}(\log_2(n)*n)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n; cin >> n; 
    vector<int> a(n); 
    for(int i = 0; i < n; i++) cin >> a[i]; 
    sort(a.begin(), a.end()); 
    int an = n; 
    for(int i = 0; i < n - 1; i++) if(a[i] == a[i + 1]) an--; 
    cout << an; 
}
`}
