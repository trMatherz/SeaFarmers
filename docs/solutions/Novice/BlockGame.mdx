# Block Game Solution

Tags: Functions

# Breakdown

For set of words, we add whichever word has a higher count of letter $l$ to the number of blocks for $l$. 

Note: if we have a character $c$, we can subtract its distance from 'a' to get the position in the vector. 

This runs in $\mathcal{O}(n)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>

{`#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
std::ifstream cin ("blocks.in");
std::ofstream cout ("blocks.out");
#include <algorithm>
using namespace std;

int get(string z, char c) { // number of times c appears in z
    int ret = 0; 
    for(int i = 0; i < z.size(); i++) if(z[i] == c) ret++; 
    return ret; 
}
int main() {
    int n; cin >> n; 
    vector<int> ans(26, 0); 
    for(int i = 0; i < n; i++) {
        string x, y; 
        cin >> x >> y; 
        for(char c = 'a'; c <= 'z'; c++) ans[c - 'a'] += max(get(x, c), get(y, c)); 
    }
    for(int i = 0; i < 26; i++) cout << ans[i] << endl;
}
`}
