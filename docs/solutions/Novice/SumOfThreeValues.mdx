# Sum Of Three Values Solution

Tags: 2P

# Breakdown

If we try to make a sum that involves $i$, then we can use two pointers for the other two values. $l < i$ and $r > i$. 

This runs in $\mathcal{O}(n^2)$

        for(int l = 0; l < i; l++) {
