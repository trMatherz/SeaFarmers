# Sum Of Two Values Solution

Tags: Sets

# Breakdown

For each $i$, we can check if there exits an element at $x - a$<sub>$j$</sub> where $j < i$. 

This runs in $\mathcal{O}(\log_2(n)*n)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
#include <map> 
using namespace std;

int main() {
    int n, k; cin >> n >> k; 
    map<int, int> v; 
    for(int i = 0; i < n; i++) {
        int x; cin >> x; 
        if(v.count(k - x)) {
            cout << v[k - x] + 1 << " " << i + 1 << endl; 
            return 0; 
        }
        v[x] = i; 
    }
    cout << "IMPOSSIBLE"; 
}
