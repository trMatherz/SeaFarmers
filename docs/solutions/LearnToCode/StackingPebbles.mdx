# Stacking Pebbles Solution

Tags: Loops

# Breakdown

We use a for loop to read-in $n$ numbers and add them. 

This runs in $\mathcal{O}(n)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;


int main() {
    int n; cin >> n; 
    int sum = 0;
    for(int i = 0; i < n; i++) {
        int h; cin >> h; sum += h; 
    }
    cout << sum;
    return 0; 
}
`}
