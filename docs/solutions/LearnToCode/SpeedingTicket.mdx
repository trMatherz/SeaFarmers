# Speeding Ticket Solution

Tags: Vectors

# Breakdown

Let $L$ be the length of the road. We can check the Bessie's speed and the speed limit at each point. 

This runs in $\mathcal{O}(L)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
std::ifstream cin ("speeding.in");
std::ofstream cout ("speeding.out");
#include <algorithm>
using namespace std;

int main() {
    vector<int> limit(101); // Road from 1 to 100
    int n, m; 
    cin >> n >> m; 
    int pos = 1; 
    for(int i = 0; i < n; i++) {
        int x, y; 
        cin >> x >> y; 
        for(int j = 0; j < x; j++) {
            limit[pos] = y; // Set limit at pos 
            pos++; 
        }
    }
    pos = 1; 
    int an = 0; 
    for(int i = 0; i < m; i++) {
        int x, y; 
        cin >> x >> y; 
        for(int j = 0; j < x; j++) {
            an = max(an, y - limit[pos]); // Check speed against limit
            pos++; 
        }
    }
    cout << an; 
}
`}
