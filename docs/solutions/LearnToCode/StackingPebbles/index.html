Stacking Pebbles Solution

Tags: Loops

Breakdown

We use a for loop to read-in nnn numbers and add them.

This runs in O(n)\mathcal{O}(n)O(n)

Implementation

#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;


int main() {
  int n; cin >> n; 
  int sum = 0;
  for(int i = 0; i < n; i++) {
      int h; cin >> h; sum += h; 
  }
  cout << sum;
  return 0; 
}
