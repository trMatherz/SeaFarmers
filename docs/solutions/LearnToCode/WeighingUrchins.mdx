# Weighing Urchins Solution

Tags: Vectors

# Breakdown

We can check each pair of urchins. 

This runs in $\mathcal{O}(n^2)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;


int main() {
    long long n, w; cin >> n >> w;  
    vector<long long> v(n); 
    for(int i = 0; i < n; i++) cin >> v[i]; 
    long long an = 0; 
    for(int i = 0; i < n; i++) {
        for(int j = i + 1; j < n; j++) {
            if(w == v[i] + v[j]) an++; 
        }
    }
    cout << an; 
    return 0;
}
`}
