# Promotion Counting Solution

Tags: If-Statement

# Breakdown

We can break this up into two cases. 

The first case has $y >= x$: 

If the row is even, then the answer can be seen to be $y^2 - x + 1$.

If the row is odd, then the answer can be seen to be $(y-1)^2 + x$.

The other case is the opposite of the first. 

This runs in $\mathcal{O}(1)$
