# Beyond The Gates Solution

Tags: Vectors

# Breakdown

Store the names in a vector. Read in all positions and output the name at that position. 

Remeber that vectors are zero-indexed!

This runs in $\mathcal{O}(n)$

# Implementation

#include <cstdio>

#include <iostream>

int main() {
    int n;

    vector<string> names(n);

}
