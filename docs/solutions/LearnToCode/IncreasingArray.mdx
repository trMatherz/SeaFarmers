# Increasing Array Solution

Tags: Max

# Breakdown

For each interger $i$ ($i > 0$), if the $a$<sub>$i - 1$</sub> $>$ $a$<sub>$i$</sub> then we increase the answer by 
the difference. 

Then, set $a$<sub>$i$</sub> = $max(a$<sub>$i - 1$</sub> $>$ $a$<sub>$i$</sub>$)$. 

This runs in $\mathcal{O}(n)$

# Implementation

{`#include <cmath>
#include <cstdio>
#include <vector>
#include <iostream> 
#include <algorithm>
using namespace std;



int main() {
    long long an = 0; 
    int n, prev;
    cin >> n >> prev; 
    for(int i = 1; i < n; i++) {
        int x; cin >> x; 
        an += max(prev - x, 0); 
        prev = max(prev, x); 
    }
    cout << an; 
}
`}
