# What's My Age Again? Solution

Tags: Input and Output

# Breakdown

We have to compare $3$ things. To stary organzied, $a, b, c$ will represent the names and $x, y, z$ will represent
their respective quantities. 

Note: It is guranteed that one of them is the biggest. 

This runs in $\mathcal{O}(1)$
