# Maximum Distance Solution

Tags: Pairs, Max

# Breakdown

We go through all pairs, maintaining the current maximum. We can store the information in a pair and just take
the max of two pais. 

Note: Pake sure the integer is the first element since this is the one that we wish to compare. 

This runs in $\mathcal{O}(n^2)$
