# Promotion Counting Solution

Tags: If-Statement

# Breakdown

If you want a naive solution and implementation, check the official solution.

For each level $i$, let $s$<sub>$i$</sub> be the starting amount and $e$<sub>$i$</sub> be the ending amount.

$e$<sub>$i$</sub> $-$ $s$<sub>$i$</sub> is the amount that needs to promote from level $i - 1$. 

This makes sense if $e$<sub>$i$</sub> $>=$ $s$<sub>$i$</sub> but if it is less, then it will cancel out the 
kind of "prefix-sums" behind my implementation. 

This runs in $\mathcal{O}(1)$
