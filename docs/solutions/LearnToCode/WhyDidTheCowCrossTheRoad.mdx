# Why Did The Cow Cross The Road Solution

Tags: Vectors

# Breakdown

We can simulate this process using a vector to keep track of which side of the road a cow is on. 
If we havn't seen the cow, they are at -1. 

This runs in $\mathcal{O}(n)$

# Implementation

<div style={{ transform: 'translateX(150px)' }}>
  <SyntaxHighlighter language="cpp" style={okaidia}>
{`#include <cmath>
#include <cstdio>
#include <vector>
#include <fstream>
std::ifstream cin ("crossroad.in");
std::ofstream cout ("crossroad.out");
#include <algorithm>
using namespace std;



int main() {
    int n; cin >> n; 
    int an = 0; 
    vector<int> a(10, -1); 
    for(int i = 0; i < n; i++) {
        int x, y; 
        cin >> x >> y; 
        x--; 
        if(a[x] != -1 && a[x] != y) an++; // check if on the other side
        a[x] = y; 
    }
    cout << an;
}
`}
